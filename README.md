# Public Hiring Challenge Sandbox

This repo is a sandbox source repo for candidate challenge packs.

Available packs:
- `challenge-packs/backend`
- `challenge-packs/devops`

The private sandbox grader provisions candidate repos by copying one of these packs into a fresh private repository. This keeps the system extensible for future `frontend`, `fullstack`, or `qa` packs without requiring a separate public template repo per role.

This root repo is not itself the candidate-facing challenge.
